
### 🛒 Lista de compras com sincronismo em nuvem

-----

ℹ️ Este é um projeto funcional para aperfeiçoar meus conhecimentos.

Tecnologias:
Kotlin w/ Coroutines
MVVM
Room
Jetpack components

🔥 Por vir:
- Login com Google
- Sincronismo de dados na nuvem
- Sincronismo entre dispositivos em tempo real

Nota: Trabalho nesse projeto no meu tempo livre por tanto, novos updates demoram alguns dias para sair.
